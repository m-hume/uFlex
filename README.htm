Updating your curent uFlex file

If you have edited the class options...

Inside the uFlex Class there is a constant declared named salt (uFlex::salt). It is strongly adviseble that you change its value as it will make your authentication systen unique. If you did changed it you will have to migrate the ::salt constant to the new class file.

As of v.51 the usual uFlex zip package now includes an 'updater' folder which includes a script(uFlex.updater.phps) that will will automatically migrate the value of uFlex::salt along with all other customizable variables inside the class that you have changed. You can always manually migrated what you have changed.

If you have NOT touched(edited) the class options...

Just replace your old class.uFlex.php file with the new one.

That Should be it.
